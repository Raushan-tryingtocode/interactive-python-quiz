#Interactive Python Quiz Game

A robust Command Line Interface (CLI) quiz application built with Python, featuring real-time input validation and dynamic scoring.

## Features
1.Input Validation: Prevents crashes by ensuring the user only enters A, B, C, or D.
2.Data Architecture: Utilizes a 'List of Dictionaries' to manage questions and answers, ensuring high maintainability.
3.Real-time Feedback: Instantly informs the user of the correct answer upon a wrong guess.

## Technical Skills Demonstrated
1.Encapsulation: Grouped related data into structured dictionary objects.
2.Control Flow: Implemented nested loops and conditional logic.
3.Version Control: Managed development via Git and GitHub (including refactoring history).

## How to Run
1. Install python.
2. Open an editor(like vs code or PyCharm)
3. Clone the repository:
   'git clone https://github.com/Raushan-tryingtocode/interactive-python-quiz.git'
4. Run the script:

   'python quiz.py'
